Creator: DusToTail

Description: This is a Unity project that tries to model an RTS game (Starcraft) regarding its systems.
These systems include:
+ Pathfinding (using flow field)
+ Select and Command System
+ Unit / Building trees
+ UI
+ Fog of War
+ AI System
+ Multiplayer System
etc

Current potential issues: performance
Potential solution: C# Job System, Async programming
